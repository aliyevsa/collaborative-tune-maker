About Collaborative Tune Maker

It is a web application where people can collaboratively make music. The main features are:

The application requires creating an account.

Users are able to create and store multiple projects.

Users are able to invite another user to a music project, and work on it with them.

Any edit to a project is saved, and also sent to another user working on the same project instantly, if there is one.

Each project has its own section for discussing the project, so that the discussions are organised around projects.

Users are able to play and stop the song on their end, individually.

The design is welcoming for a wide range of users.

Users can share the URL of their project so that anyone can view and listen to it.
